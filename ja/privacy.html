2. 収集する情報

ユーザーから直接提供される情報

  • メールアドレス (任意) — ウェイトリスト登録、アップデート購読、メールマジックリンク認証をご利用の場合のみ。
  • Discord ハンドル (任意) — ウェイトリストフォームで自発的に提供され、コミュニティサーバー招待に使用します。
  • アライアンス名簿 CSV と座標 — 該当ツールにアップロードされた場合。アライアンス署名(R5 名 + 上位 8 メンバーの SHA-256 ハッシュ)ごとに保存され、IP アドレスとは分離されます。
  • Discord Webhook URL — ツールがアライアンスチャットに自動投稿できるよう貼り付けられた場合。
  • サポート要求の内容 — フィーチャーリクエストや問い合わせウィジェットに入力された内容。

自動収集される情報

  • IP アドレス — レート制限のため短期間保持(最大 1 時間)、その後は分析目的で /24 ネットワークに匿名化。
  • ブラウザ言語、国(IP から推定)、ユーザーエージェント — UI ローカライズおよび集計分析用。
  • ページビュー・イベント — Plausible(クッキー不使用)および Google Analytics 4(IP 匿名化オン、Google Signals オフ)で収集。
  • 紹介元アトリビューション — `?ref=CODE` リンクからアクセスされた場合、コードはファーストパーティ Cookie に保存(第 5 条参照)。

決済プロセッサー(Stripe)が収集する情報

  • Stripe 決済メタデータ — メールアドレス、カード名義、請求国、カード番号下 4 桁、取引金額。カード番号全体を閲覧・保存することはありません。Stripe は自社のプライバシーポリシーに従って処理: stripe.com/privacy。

4. 保管期間

  • ニュースレター購読者: 解除まで保管、その後は再送防止目的でさらに 2 年間。
  • Stripe 決済記録: 米国税法遵守のため 7 年間。
  • 分析データ: 26 か月(Google Analytics 4 デフォルト)。Plausible は集計データのみで生データは保持しません。
  • ツールにアップロードされた名簿/座標: 最終利用後 90 日で自動削除。
  • IP アドレス: レート制限用に 1 時間、その後匿名化。
  • サポートチケット: 3 年間、その後は識別子を除去してアーカイブ。

Cookie は最小限に抑えています。Plausible は完全に Cookie 不使用。Google Analytics 4 は `client_id` を Cookie ではなくブラウザの `localStorage` にのみ保存する設定です。

6. 第三者との共有

各ベンダーが業務を遂行するのに必要な最小限のデータのみ共有します。データを販売しません。広告目的で共有しません。

  • Stripe — 決済処理。プライバシーポリシー。
  • Plausible Analytics — Cookie 不使用サイト分析。EU 拠点、GDPR 準拠。プライバシーポリシー。
  • Google Analytics 4 — ページビュー集計。IP 匿名化オン、Google Signals オフ。プライバシーポリシー。
  • SMTP プロバイダ — トランザクションおよびニュースレターメール送信(現在 Postmark / Amazon SES)。共有データ: メールアドレスとメッセージ内容。
  • Railway — バックエンド API ホスティング。米国インフラ。プライバシーポリシー。
  • GitHub Pages — 静的サイトホスティング。プライバシーポリシー。
  • Cloudflare — DNS とエッジキャッシング。プライバシーポリシー。
  • Anthropic (Claude API) — KB チャットと名簿抽出機能に使用。質問または名簿フレームのみ送信され、アカウント情報は送信されません。プライバシーポリシー。

Stripe、Plausible、Anthropic とはデータ処理契約(DPA)を締結済みです。他のベンダーとの正式な DPA も進行中で、締結次第リンクを掲載します。

7. 国外移転

ホスティングは主に米国です。EEA、英国、韓国、日本にお住まいの場合、データは米国に移転・処理されます。以下の保護措置を講じます:

  • ベンダーが自己認証している場合の EU-U.S. データプライバシーフレームワーク(Stripe、Google、Cloudflare、GitHub)。
  • その他の移転については、委員会決定 2021/914 に基づく標準契約条項(SCC)。
  • 追加技術的措置: 転送中の暗号化(TLS 1.2 以上)、保存時の暗号化、国外に送信される個人データの最小化。

10. セキュリティ

  • ブラウザとサーバー間の全通信は TLS 1.2 以上を使用。
  • アクセスコードは平文ではなくハッシュで保存。
  • 名簿アップロードは IP アドレスと分離されたアライアンス署名で保存。
  • プロフィールスタジオで処理される写真は端末を離れません。
  • バグバウンティポリシーは /ja/trust.html、責任ある脆弱性開示を歓迎します。

完璧に安全なシステムはありません。個人情報に関わる侵害が発生した場合、GDPR 第 33 条(および該当法令の同等規定)に従い 72 時間以内に通知します。

11. 自動化された意思決定

あなたに法的または同等の重大な影響を及ぼす自動化された意思決定に個人データを使用しません。AI 機能(KB チャット、名簿抽出機能)はテキスト生成やデータ抽出を行いますが、あなたに関する意思決定は行いません。
