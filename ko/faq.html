시작하기

Q1R5TOOLS가 뭔가요?

R5TOOLS.IO는 Last War: Survival 얼라이언스 R5·R4를 위한 11개 계획 도구 모음입니다. 대부분의 얼라이언스가 여전히 사용 중인 "구글 시트 + 디스코드 PDF" 워크플로우를 대체해 드립니다. 대표 도구는 로스터 추출기로, 얼라이언스 멤버 목록을 90초간 화면 녹화해서 업로드하시면 Claude 비전 AI가 모든 행을 CSV로 자동 변환해 드립니다 (약 2분 소요).

CSV가 생성되면 나머지 모든 도구 — 랜딩 플래너, 열지도 시뮬레이터, 동결 위험 대시보드, 석탄 계산기, 도시 점령 스케줄, 시즌 타임라인, 하이브 그리드, 프로필 스튜디오, VS 데이 최적화, KB 챗 — 이 모두 그 CSV를 그대로 사용합니다. 7개 언어 지원 (한국어 우선), 모든 도구가 디스코드용 PNG 내보내기 지원. 현직 R5가 직접 만들었습니다.

Q2어떻게 잠금해제 하나요?

access-codes.r5tools.io 에 접속해서 코드를 입력하시면 됩니다. 한 번 입력하면 90일간 그 기기의 모든 도구가 잠금해제됩니다. 세 가지 방법:

  1. 무료 — 워존 2007 멤버: R5에게 RONY-FREE 코드를 요청하세요 (실질적으로 어느 워존이든 사용 가능).
  2. $10 개인 요금제: Stripe로 1회 결제, 개인용 전체 도구 사용.
  3. $50 얼라이언스 번들 (또는 $30 파운딩 — 얼라이언스 10곳 한정): R5가 1회 결제 후 코드를 얼라이언스 전체에 공유.

쿠키를 지우거나 새 기기에서 사용하실 때는 같은 코드를 다시 입력하시면 됩니다.

Q3RONY-FREE가 뭔가요?

RONY-FREE는 RONY 얼라이언스 멤버와 Evan이 직접 초대한 한국어 사용자를 위한 주 무료 코드입니다. 시스템상 워존 2007로 표시되지만 실제로는 어디서든 사용 가능합니다. access-codes.r5tools.io 에 입력하시면 90일간 11개 도구 전체를 그 기기에서 사용하실 수 있습니다.

얼라이언스 내 다른 분이 이미 쓰고 계시다면 그분께 여쭤보세요 — 같은 코드로 얼라이언스 전체가 사용 가능합니다. RONY 소속이 아니더라도 기존 멤버가 안내해 드렸다면 자유롭게 사용하셔도 됩니다.

Q4설치할 것이 있나요?

없습니다. 모든 도구가 브라우저에서 바로 실행됩니다. 앱, 다운로드, 빌드 과정 전부 없습니다. KB 챗과 로스터 추출기만 서버(Claude API)를 호출하고, 나머지 9개 도구는 완전히 클라이언트 측에서 작동합니다 — 여러분의 로스터 CSV는 명시적으로 업로드하지 않는 한 여러분 기기에서 벗어나지 않습니다. r5tools.io를 즐겨찾기에 추가하시면 끝입니다.

Q5스마트폰에서도 작동하나요?

대부분의 도구는 작동합니다. 데스크톱과 iOS의 Chrome, Safari에서 테스트합니다. 로스터 추출기는 아이폰 .mov와 안드로이드 .mp4 화면 녹화를 지원합니다 — 대부분의 R5분들은 스마트폰 하나로 전체 과정을 진행하십니다. 무거운 계획 도구(열지도, 하이브 그리드)는 화면이 넓은 태블릿이나 노트북이 편합니다. Firefox도 보통 잘 작동합니다. 구형 안드로이드는 기기마다 다릅니다 — 문제가 있으시면 페이지 하단의 💡 피드백 버튼으로 브라우저 정보와 함께 알려주세요.

가격 및 환불

Q6$10 / $30 / $50이 각각 뭘 제공하나요?
  • $10 개인 (1회 결제): 11개 도구 전체를 개인 1명이 90일간 사용. 로스터 추출기 표준판 포함.
  • $30 얼라이언스 파운딩 (얼라이언스 10곳 한정, 이후 $50): 얼라이언스 전체가 코드 하나를 공유. 평생 신규 도구 무료 사용 추가. 10자리 한정.
  • $50 얼라이언스 번들 (1회 결제): 파운딩과 동일한 얼라이언스 공유 방식, 평생 혜택만 제외. 5명이 개인 코드 각자 사는 것보다 저렴.
  • +$5 로스터 추출기 프리미엄: 애드온. Claude 2차 감사 실행 — 94/94 완벽 정확도 보장.

전부 1회 결제입니다. 구독 아님, 자동 갱신 아님.

Q7환불되나요?

가능합니다 (terms.html 기준):

  • 결제 후 24시간 이내, 코드 미사용: 무조건 전액 환불. hello@r5tools.io 로 메일 주세요.
  • 코드 이미 사용: 환불 불가. 단, 저희 측 문제로 도구가 48시간 이상 연속 다운된 경우 예외.
  • 얼라이언스 분쟁: 얼라이언스가 해체되어도 결제한 R5에게 환불하지 않습니다. 대신 코드를 후속 얼라이언스에 넘겨주실 수 있습니다.

저희는 1인 개발 팀입니다. 솔직하게 사정 말씀해 주시면 대부분 협의해 드립니다.

Q8파운딩 요금제는 왜 마감되나요?

파운딩 요금제는 가장 먼저 참여하는 얼라이언스 10곳을 위한 평생 특가입니다: $30 1회 결제 (정가 $50 대비) + 앞으로 나올 모든 신규 도구 평생 무료. 이 10명의 R5가 저희 제품이 계속 개발할 가치가 있다는 것을 검증해 주시는 분들이고, 이분들의 결제금이 향후 6개월간의 개발을 지원합니다.

10곳이 채워지면 파운딩은 가격 카드상 마감됩니다. 실시간 남은 자리는 r5tools.io#pricing 에서 확인하실 수 있습니다.

Q9얼라이언스가 비용을 나눠 낼 수 있나요?

가능합니다 — 정확히 그 방식으로 설계된 것이 얼라이언스 번들($50)과 파운딩($30)입니다. R5 한 분이 1회 결제하시고, 그 코드를 얼라이언스 전원에게 공유하시면 됩니다. 각 멤버가 access-codes.r5tools.io 에서 같은 코드를 입력하면 그 기기에서 90일간 사용 가능합니다. 흔한 사례:

  • R5가 결제 후 얼라이언스 자금에서 정산.
  • 임원 5명이 $50을 균등 분담 (1인당 $10).
  • 상위 파워 4명이 $10씩 분담해 번들 구매.

Stripe 결제 시 메모란도 사용 가능하니 개인 코드를 선물로 드리셔도 됩니다.

추천 프로그램

Q10제 추천 코드는 어떻게 받나요?

r5tools.io/refer.html 로 접속하셔서 이메일로 로그인하세요 — 매직 링크 방식이라 비밀번호가 없습니다. 로그인하시면 여러분 전용 코드(R5-닉네임-A7K3 형식)와 공유 링크를 확인하실 수 있습니다. 이미 익명 코드로 잠금해제하셨더라도, 추천 프로그램 참여는 이메일 로그인이 별도로 필요합니다 — 그래야 수익을 정확히 여러분께 크레딧해 드릴 수 있습니다.

Q11언제 정산 받나요?

정산은 수동입니다 — Evan이 매주 CSV를 뽑아 PayPal 또는 CashApp로 각 추천인께 지급합니다. 최소 금액 없음, 스케줄 눈치 게임 없음, 크레딧 회수 없음. 추천 링크를 통해 30일 이내에 유료 결제가 발생하면 총액의 25%를 크레딧해 드립니다 (first-touch 기준 — 먼저 태그된 사람이 획득). 자기 추천은 차단됩니다. 원장은 확인 가능 — hello@r5tools.io 로 요청하시면 여러분의 행을 보내드립니다.

EVAN — 결정 필요 한국 추천인 정산 방법 (PayPal · Wise · 계좌이체?)과 최소 지급 금액 결정 필요.

Q12YouTube · TikTok · 트위치에서 r5tools를 홍보해도 되나요?

당연히 됩니다 — 오히려 환영합니다. 추천 링크(r5tools.io?ref=R5-닉네임-XXXX)를 설명란에 넣으시면 30일 이내 구매자에 대해 25% 크레딧됩니다. 지침:

  • r5tools가 First Fun · Last War: Survival과 제휴 관계라고 주장하지 마세요 — fan-made 도구입니다.
  • 저희가 하지 않는 것을 약속하지 마세요 (예: "밴 절대 없음 보장", "자동 플레이" — 게임 자동화는 하지 않습니다).
  • 저희 도구 화면 녹화는 자유롭게 가능. 게임 자체 녹화는 First Fun의 ToS와 여러분 간의 문제입니다.

EVAN — 결정 필요 검증된 크리에이터(구독자 X명 이상)에게 더 높은 rev share를 주는 크리에이터 프로그램을 정식화할지 결정 필요.

한국어 · 국제화

Q13한국어 지원되나요?

네, 완벽 지원합니다. 저희는 한국어를 1순위 언어로 개발했습니다 (영어보다 우선). 모든 도구, 랜딩 페이지, PNG 내보내기까지 100% 한글화되어 있고, 워존 2007 한국어 사용자가 주 사용자층입니다. RONY(우군 얼라이언스) R5가 무료 코드 RONY-FREE를 배포하고 있으니 R5에게 요청하세요. 카톡·디스코드 어디서든 코드 공유 가능합니다.

Q14한국어와 영어 외 다른 언어도 지원하나요?

네 — 첫날부터 7개 언어: 한국어 · English · Español (스페인어) · Português (포르투갈어 브라질) · 日本語 (일본어) · Deutsch (독일어) · Français (프랑스어). 페이지 우측 상단에서 전환하실 수 있습니다. 모든 도구, 모든 버튼, 내보내는 모든 디스코드 PNG가 선택하신 언어로 렌더링됩니다.

번역은 가능한 한 원어민 R5가, 나머지는 Claude Sonnet 4.6 + 사람 검수로 진행되었습니다. 어색한 부분이 있으면 💡 피드백 버튼으로 알려주세요 — 몇 시간 내에 수정합니다.

기술 문제

Q15도구가 안 열려요 · 오류가 나요.

먼저 세 가지 빠른 해결책:

  1. 강력 새로고침: Cmd+Shift+R (맥) 또는 Ctrl+F5 (윈도우) — "고장났다"는 신고의 대부분은 캐시된 낡은 JS 때문입니다.
  2. 코드 재입력: 쿠키는 90일 후 만료됩니다. access-codes.r5tools.io 에서 다시 입력하세요.
  3. 다른 브라우저 시도: Chrome, Safari에서 테스트합니다. Firefox도 보통 잘 됩니다. 구형 안드로이드와 앱 내 브라우저(디스코드, 인스타)는 편차가 큽니다.

여전히 안 되시면 💡 피드백 보내기 버튼을 눌러주세요 — 도구 이름, 브라우저, 참조 ID와 함께 Evan의 디스코드로 즉시 전달됩니다. 긴급 문제는 빨강 표시되어 @here 알림이 갑니다. 상태 페이지: r5tools.io/status.html.

Q16로스터는 어떻게 가져오나요?

로스터 추출기 (roster.r5tools.io)를 사용하세요. 3단계:

  1. 게임에서 얼라이언스 멤버 목록을 여세요. iOS · Android 화면 녹화 시작. R5부터 R1까지 천천히 스크롤 (100명 얼라이언스 기준 약 90초). 녹화 중지.
  2. 생성된 .mov 또는 .mp4 파일을 업로드.
  3. Claude 비전이 모든 행을 자동 판독합니다. 약 2분 내에 CSV + JSON 완성. API 비용 약 $0.45.

이 CSV가 나머지 모든 도구의 입력이 됩니다. 완벽한 정확도가 필요하시면 로스터 추출기 프리미엄 (+$5)을 추가하시면 2차 감사로 94/94 정확도를 보장해 드립니다.

Q17여러 기기에서 사용할 수 있나요?

네. 쿠키가 기기별이라 노트북, 스마트폰, 태블릿에서 각각 잠금해제하시면 각 기기가 각자 90일 쿠키를 갖게 됩니다. 좌석 제한 없음. 얼라이언스 번들 · 파운딩 코드는 명시적으로 얼라이언스 전체 기기 공유 목적입니다. 개인 $10 코드는 원래 1명용이지만 기기 잠금은 없습니다 — 신뢰 기반입니다.

EVAN — 결정 필요 기기당 · 계정당 제한을 둘지 결정 필요. 현재 codes.json에는 제한이 없음.

신뢰 및 안전

Q18Last War: Survival · First Fun과 관계가 있나요?

아니요. R5TOOLS.IO는 팬 제작 독립 도구 모음입니다. First Fun Group, Century Games, Last War: Survival 과 어떤 형태의 제휴, 승인, 라이선스 관계도 없습니다. 모든 상표와 게임명은 각 소유자의 재산입니다. 저희가 이 도구들을 만든 이유는 오직 저희가 게임을 하고 이런 도구가 필요했기 때문입니다. First Fun이 어떤 변경을 요구하시면 즉시 응할 것입니다. 전체 면책 조항: terms.html.

Q19이거 쓰면 계정 밴 당하나요?

0% 리스크를 보장할 수는 없습니다 (정직한 누구도 그럴 수 없습니다). 하지만 저희 도구는 First Fun ToS의 안전한 쪽에 남도록 설계되어 있습니다:

  • 어떤 게임 내 액션도 자동화하지 않습니다. 자동 플레이 없음, 자동 공격 없음, 봇 이동 없음, 패킷 인젝션 없음, 게임 클라이언트 수정 없음.
  • 저희는 계획 도구입니다 — 여러분이 직접 실행하십니다. 화면 녹화도 여러분이 수동으로 하시고, 저희는 랜딩 계획, 석탄 계산, 하이브 배치 등을 저희 쪽에서 도와드립니다. 실제 실행은 여러분이 게임 내에서 수동으로 하십니다.
  • 여러분의 게임 계정 정보를 절대 취급하지 않습니다. 로그인 정보 요구 없음. 저희가 대신 로그인할 수 없음.

다만: LWS ToS는 "부당 이득을 제공하는" 서드파티 도구를 금지합니다. 구글 시트, 석탄 계산기, 하이브 배치도 같은 계획 도구는 LWS 커뮤니티에서 광범위하게 사용되고 있고 (저희가 아는 한) 이로 인한 밴 사례는 없습니다. 저희는 이를 계속 모니터링하며 ToS 해석이 바뀌면 업데이트할 것입니다.

Q20제 데이터는 어떻게 처리하나요?

간단 요약 (전체 정책: r5tools.io/privacy.html):

  • 여러분의 로스터 CSV는 "얼라이언스 시그니처"(R5 이름 + 상위 8명 멤버의 SHA-256 해시) 기준으로 저장되며, IP와 분리됩니다. 다른 얼라이언스와 절대 공유 안 함. 판매 안 함. 모델 학습에 사용 안 함.
  • 디스코드 웹훅 URL은 여러분의 액세스 코드 하에 저장되어 자매 도구 간 자동 동기화됨. 다른 얼라이언스에 노출 없음.
  • 분석 데이터는 집계 + IP 마스킹(마지막 옥텟 0으로). Plausible(쿠키 없음)과 선택적 GA4(anonymize_ip: true, 광고 개인화 없음) 사용.
