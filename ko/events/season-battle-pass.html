  • 04-squads-combat — Tactics Cards debuted in S4; card meta shifted hard in S6 with the removal of the 4+1 hybrid squad strategy from 글로벌 익스페디션, forcing pure 탱크/미사일/항공기 formations. Curse Resistance / 바이러스 저항 (VR) stats are new gate mechanics for PvE damage that stack over standard stats.
  • 05-season-1-crimson-plague — S1 크림슨 플레이그 introduced T10 troops. 시즌 structure since S3 has become more elaborate but retains the S1 template of themed map + seasonal resources + 수도/faction endgame.
  • 06-season-2-frozen — S2 폴라 스톰 was the first true themed environmental hazard (cold/heat). S3+ retained the "seasonal building trinity" model (resource-producer + shop-currency-producer + resistance-lab). 얼라이언스 화로 pattern from S2 evolves into 축복의 샘 (S3) / 등대 (S4) / 은행 요새 (S5) / 제단s (S6). See STYLE_GUIDE for how the 라워_얼라이언스_Ops_Center suite handles seasonal building overlays.
  • 08-얼라이언스-systems — 얼라이언스 협약s (S6), 얼라이언스 세이프 타임 (S5+), and 파벌 기술 (S6) reshape how 얼라이언스s coordinate at the meta level. 얼라이언스 city/stronghold caps evolve per season (S3 = 8/8; S4 = 6/6; S5 Bank = 4 default up to 12; S6 factional).
  • 09-events — 블러드 나이트 강림 (S4, 02:30/10:30/18:30 server), 모래벌레 위기 (S3, 14 days), 향신료 전쟁 (S3, biweekly), 구리 전쟁 (S4, Wed/Sat), 무역 열차 (S5), 제단 Tuesdays (S6), 파벌 대전 (season-ending). Each season adds 2–3 new recurring events.
  • 10-economy — 시즌 Battle Pass (~48 days, ~€11.99 Advanced / ~€23.99 Luxury) is the shipped pricing anchor since S3. Weekly Pass adds a temporary seasonal building + resources. Three exclusive weapon passes per season (Weeks 1/3/6). 히어로 각성 shards are now a paid axis on top of exclusive weapons.
  • 14-ui-reference — 시즌 transitions require map re-render, seasonal building rendering, and faction color coding (S6 Deepwood green / Wetland blue / Great River neutral). 얼라이언스 세이프 타임 UI added in S5.
  • 15-glossary — Every season has full KR terminology; namu.wiki maintains a running 한국어-language reference. Faction names (카게노사토, 코부타이 in S4; Deepwood/Wetland in S6) are transliterated from EN in KR sources rather than re-localized.
