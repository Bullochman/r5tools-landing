지휘관 최고봉 / Strongest 총사령관 — one-time server-launch 7-day sprint

성ts a couple of days after server launch. 7 days, points reset daily 00:00 server time. 7 stages of themed activities:

  • Stage 1: recruit heroes + spend hero EXP (max 10,000 pts in 5 tiers: 500/1k/3k/6k/10k)
  • Stage 2: construction speedups + resource gathering
  • Stage 3: research speedups + gathering
  • Stage 4: stamina consumption + gathering
  • Stage 5: training speedups + gathering
  • Stage 6: enemy unit kills (L1–L10 = 2–17 pts) + healing acceleration
  • Stage 7: any Stage 1–6 activity scores

Ranking rewards for top cumulative scorers via mail.

Foundation Day / 서버 Anniversary: No widely-documented separate 라워 event with the exact name "Foundation Day." The nearest analog is either the 지휘관 최고봉 launch event (new server) or the New Year / seasonal anniversary bundles. Flagged as an open question — may be region-specific naming for what other guides call the anniversary Battle Pass or a KR-market localized event.
