윈터 스톰 배틀필드 — 5v5 arena mode

5-minute matches, 5v5, on a dedicated battlefield. Base level 15+ required. Three capture points: 미사일 Station (center, opens after 2 min, "massive firepower"), MG Site (1 per team), Artillery Site (1 per team). Weekly Honor 연맹 코인 (up to 10,000/week cited) for legendary gear.

Victory conditions: eliminate all opposing Infantry + 장갑ed Forces, OR have more surviving forces when time expires. Medical drops enable in-battle healing. Up to 3 wins and 3 losses per event cycle yield rewards; typically Sunday, starting week 4 of a season.

이벤트 무과금 should skip or minimize:

  • Hot Deal packs (except free-track redemption).
  • 반짝이는 시장 cosmetics — every source flags this as a P2W trap.
  • Bounty Hunter paid-bullet track (2,000 diamonds/day for 80 diamond-bullets) — free 20 bullets/day still lets 무과금 clear meaningful voucher totals.
  • Desert Storm / Winter Storm if not top-32-얼라이언스 eligible (Desert Storm) or if squad-power is too low to win 3 matches (Winter Storm point system now compensates, see revised section).

Winter Storm — new point-based reward system (added 2026-07-15)

The Sunday 5v5 윈터 스톰 배틀필드 reward system changed per [cpt-hedge.com/guides/winter-storm-reward-structure-changes WebSearch excerpt, 2026-07-15, 티어 B]. Prior KB claim was "10,000 Honor 연맹 코인/week cap requires 3 wins." That is now outdated.

New (post-update) system:

  • Total Honor 연맹 코인 across all 6 chests: 16,500 (up from 10,000 in the win-count system)
  • 연맹 코인-based, not win-count-based — every match scored contributes regardless of win/loss, so a run of losses with high objective-scoring still unlocks rewards
  • First reward chest unlocks at 45,000 points; final chest at 450,000 points
  • Old system still active in parallel: 3 victories still grant 10,000 Honor 연맹 코인 via the win-track (a win auto-counts as participation)
  • UR Gear Blueprints in the event shop still cost the same Honor 연맹 코인 — the new system just makes them reachable without needing 3 wins

무과금 impact: the new system removes the "no wins = no rewards" cliff that made Winter Storm feel punishing for lower-power players. Even a squad that loses every match can now hit the 45k first-chest tier via objective play.
