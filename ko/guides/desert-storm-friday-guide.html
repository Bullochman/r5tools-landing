라스트 워: 서바이벌's calendar is a stack of overlapping loops: a daily loop (무기 경쟁, six 4-hour phases), a weekly loop (얼라이언스 듀얼 VS Mon–Sat, plus rotating 좀비 시즈, 고스트 옵스, 프론트라인 브레이크스루, Bounty Hunter, Desert Storm), a bi-weekly loop (둠스데이, 좀비 인베이전, General's Trial), and the seasonal loop (8-week 시즌 with a mid-season 수도/왕국 대 왕국-equivalent war — 희귀 토양 전쟁 in S2 (see 06-season-2-frozen), 향신료 전쟁 in S3, 코퍼 War in S4, 골드vein / 골드en Palace in S5, faction war in S6 (see 07-future-seasons)). Between seasons the map opens up for off-season cross-server PvP (미티어라이트 아이언 워, 스카이 배틀프론트). Serious 얼라이언스s stack VS Day themes on top of matching 무기 경쟁 phases to score the same action twice, and reserve their biggest resource dumps for Friday (총동원) and Saturday (Enemy Buster). Preparation — pre-staged construction/research, hoarded speedups, badge stockpiles, and shielded inactive members — decides more of these events than any in-battle skill. Most timers run on server time (00:00 reset); PvP windows are typically 12:00–22:00 server time. 라워 does not currently ship a single "왕국 대 왕국" event by that exact name — the closest analogs are season-scoped 얼라이언스 wars (희토 / 향신료 / 코퍼 / 골드vein) and the off-season 스카이 배틀프론트 and 미티어라이트 아이언 워, which are the game's true inter-server PvP moments.

  1. Daily: 무기 경쟁 (24 h split into six 4-hour phases), Bosses Code, radar tasks, 둠 워커s on active days, hero-shard daily rotations.
  2. Weekly: 얼라이언스 듀얼 (VS, Mon–Sat), 고스트 옵스 (Thursday windows), 좀비 시즈 (once every 1–2 weeks), 데저트 스톰 배틀필드 (Fri), 프론트라인 브레이크스루 (Sun), Bounty Hunter (rotating), the Hot Deal / 반짝이는 시장 cycles.
  3. Bi-weekly: 좀비 인베이전 (glacieradon post-S2), 둠스데이 (Sun), General's Trial (4-day window), Bounty Hunter alternating cycles.
  4. 시즌al: 8-week season with a locked schedule of week-by-week unlocks that culminates in a mid-to-late season "big war" (희토 / 향신료 / 코퍼 / 골드vein 골드en Palace / faction war in S6). Between seasons: 스카이 배틀프론트 and 미티어라이트 아이언 워 fill the gap.

수도 / Grand Nexus / 대통령ial elections: Winning 얼라이언스's R5 becomes 대통령 and appoints Vice 대통령 + 8 Ministry positions (HQ 16+ minimum to apply). Each position gates a global server buff (attack/defense/HP, construction %, research %, resource output, hospital cap, training). Conqueror bonus (during 왕국 대 왕국/season-war windows) multiplies these ~25%. 내각 pay out via 30-minute wait between applications, 50-application cap per position, 24-hour expiration.

좀비 시즈 is the 얼라이언스 PvE co-op event where 20 escalating zombie waves attack every eligible member's base simultaneously. Waves last ~30 seconds each with a short break.

레이더 Clue 연맹 코인 are the currency that unlocks 좀비 시즈 access. Earn during cooldown windows between cycles:

  • Defeating Doom Legion / 파멸의 정예 / 둠 워커s: +300 each
  • Killing zombies, assisting allies, geological sampling, gathering, digging: +10 each

글레이시아라돈 (post-시즌 2):

  • 얼라이언스-wide boss requiring accumulated zombie-boss defeats
  • Each 얼라이언스 can start one 글레이시아라돈 challenge per event
  • Max 30 hits per commander
  • 60-minute time limit per challenge
  • New 얼라이언스 members cannot collect 글레이시아라돈 treasure within 24 hours of joining

Distinct from 좀비 시즈 and Invasion, this is a bi-weekly Sunday event where the server 얼라이언스 defeats a zombie boss that has invaded the "National Assembly / 국회." Real-time 5v5 team battle on a dedicated battle server (troop losses separate from HQ troops). Win by scoring most points in the time limit OR reaching 30,000 points first. Boss appearance order: Baseball Boss → Boxing Boss → Rugby Boss. Reward: 훈장s of Merit ("명예훈장") → dedicated store.
