우리 얼라이언스 잠금 해제 →
  1. Daily: 무기 경쟁 (24 h split into six 4-hour phases), Bosses Code, radar tasks, 둠 워커s on active days, hero-shard daily rotations.
  2. Weekly: 얼라이언스 듀얼 (VS, Mon–Sat), 고스트 옵스 (Thursday windows), 좀비 시즈 (once every 1–2 weeks), 데저트 스톰 배틀필드 (Fri), 프론트라인 브레이크스루 (Sun), Bounty Hunter (rotating), the Hot Deal / 반짝이는 시장 cycles.
  3. Bi-weekly: 좀비 인베이전 (glacieradon post-S2), 둠스데이 (Sun), General's Trial (4-day window), Bounty Hunter alternating cycles.
  4. 시즌al: 8-week season with a locked schedule of week-by-week unlocks that culminates in a mid-to-late season "big war" (희토 / 향신료 / 코퍼 / 골드vein 골드en Palace / faction war in S6). Between seasons: 스카이 배틀프론트 and 미티어라이트 아이언 워 fill the gap.

Bi-weekly event where players kill invading zombies (3 small L10 zombies per spawn — 10 courage medals + resources) and Zombie Bosses (spawn randomly after invading-zombies are cleared; boss level tied to server's summoning level).

Weekly event pair rotation: the odd/even week number since the server's current season started decides which pair runs — General's Trial + 둠스데이 on one parity, 좀비 인베이전 + Rampage Bosses on the other. This resolves an open question about why a specific week seems to skip 둠스데이 — it's the alternating parity, not a bug.

Wanted Bosses (Codes 39 / 64 / 87) — weekly-rotating hero-type-buffed boss damage race

Added 2026-07-15. Wanted Bosses are the game's rotating damage-race event in which a specific hero-type gets a +50 % damage buff. Sources: [theriagames.com/guide/last-war-survival-wanted-boss-code-39/ WebFetch 2026-07-15, 티어 B]; [lastwarvault.com/guides/general/wanted-boss-guide/ WebFetch 2026-07-15, 티어 B]; [lastwargame.online/en/bosses-code-39-64-87/ WebFetch 2026-07-15, 티어 B]; [buffbuff.com Boss Code overview via search excerpt, 2026-07-15]; [btcpiratesnation.com/wanted-boss-code/ via search excerpt, 2026-07-15].

Spawn schedule (per Vault + LastWar.online): 4 fixed windows per event day at 00:00 / 06:00 / 12:00 / 18:00 server time, each spawn staying on the map for 3 hours before despawning. Third-source corroboration 2026-07-16 via LastWar.online search-excerpt reconfirms the identical 4-spawn / 3-hour / 5-attempts / HQ-8 profile — the rotation and cadence are now triple-sourced.

Fight mechanics:

  • Each attempt lasts 1 min 30 sec — inflict as much damage as possible in the window
  • Boss does not attack you — no defensive heroes needed; pure DPS squads win
  • 5 attempts per day; solo attacks only (no 랠리 option per Vault)
  • Only your highest single-attempt damage counts for ranking rewards
  • Rewards structured as ranking rewards (top-N placement) + first-3-attacks individual rewards + 6-tier objective/goal rewards
  • Minimum HQ level 8 to participate

무과금 optimization (per Vault):

  1. Always match troop type to the day — the +50 % type bonus vastly outweighs gear-quality differences
  2. Activate 전쟁 열풍 by scouting an enemy base before attacking for a free damage boost
  3. Use cheap attack + crit potions — meaningful damage gain vs their trivial cost
  4. Never skip a day — even a single attempt per spawn window guarantees baseline rewards; long-term compounding on Upgrade 광석 + Synthetic Resin is significant
  5. Gear quality > hero star level for this event

R5 implication for tool design: since the buffed type rotates on a weekly Mon-Sat schedule (not daily), the Bosses Code Rotation Tracker tool from open-questions.md simplifies to a static 6-day calendar rather than a rolling day-N lookup — a fixed weekly-view widget is enough. 이벤트 무과금 should hit every cycle (tier-1 priority):

  1. Wanted Bosses (Codes 39/64/87) — 5 free attempts/day, no cost, Upgrade 광석 + Synthetic Resin compounds long-term. Never skip a day.
  2. 무기 경쟁 — free daily; hoard speedups + valor badges between cycles, dump them during matching phases only.
  3. 얼라이언스 듀얼 VS — Wednesday's valor-badge spend + Thursday's shard spend are the two biggest single-day contributions a free player can make. Chest 1–3 (Basic, 540k pts) is a realistic 무과금 weekly target.
  4. 고스트 옵스 — 3 assists + 5 plunders per Thursday hits 6 rewards with zero premium currency. Prefer 전용 무기 조각s over UR shards per LDShop — the weapon system is the 무과금 power multiplier that closes the gap with whales.
  5. Trade Truck plunders — 4 enemy trucks per day; six-refresh rule (refresh, plunder 1–2, wait 10–30 sec, refresh again — by the 6th refresh, a UR-quality truck is near-certain) [lw-spy.com/blog/last-war-truck-plunder-guide/ WebSearch 2026-07-15, 티어 B]. Reindeer Sleds are the highest-priority truck type — 전용 무기 조각s are cited as the single most valuable reward in the entire truck system.
  6. 반짝이는 시장 — even without diamond spend, 무과금 earns ~300–360 골드s/6-day cycle from the daily-diamond exchange (only if diamonds are saved for it, not spent on shields). 골드s do NOT expire between cycles. Stack 8–10 cycles = one full Mythic Gear Blueprint set (3,000 coins). This is the single most important long-cycle 무과금 strategy.
