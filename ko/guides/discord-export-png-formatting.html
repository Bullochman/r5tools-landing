돕기 daily task ladder (added 2026-07-16). Independent of the raw contribution-points ceiling, the 돕기 daily task list unlocks progressive rewards at 10 / 50 / 100 / 200 / 300 cumulative helps in one day. The 300-help tier is the top rung of the daily task list, which implicitly puts the daily contribution-points cap at or above 300 helps (the last tier would not be reachable otherwise). Community guides publish this ladder as tier thresholds, not a cap statement, but the 300 upper tier is the tightest published lower-bound the KB now has on the daily-help ceiling. Downstream tools (돕기 Efficiency Tracker) can safely assume ≥300 helps/day as the plannable ceiling until an in-game screenshot pushes the number higher. Format. R4 or R5 places the Marshal's Guard platform on the world map. The platform is a 3×3-tile object placed at the anchor coordinate the leader picks (the working anchor for planning tools is {x:486, y:432} on many servers, but it's leader-controlled). Five Marshal's tanks appear on the platform; the 얼라이언스 rallies them one at a time until either all five die or the 30-minute clock runs out.

  • Player HQ / city: 2×2 tiles per base footprint (per 하이브-planner tools; note some sources treat member "cells" as 3×3 in planner UIs where one cell = one base + 1-tile buffer).
  • Marshal's Guard platform: 3×3 tiles (1 event cell).
  • 연맹 센터 (world building placement varies; the on-base structure is 5-tile-footprint per planners): 5×5 tiles.
  • 미사일 silo (turret/rocket world object): 4×4 tiles.
  • 요새 seasonal fortress: 21×21 tiles (a 7-cell × 7-cell block in a 3-tile-per-cell planner grid).
