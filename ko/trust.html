법률 자문이 아닙니다. 본 페이지의 내용은 현재의 약속이며, 강제력 있는 버전은 이용약관 및 개인정보에 있습니다.

게임 스튜디오와의 무관성

r5tools.io는 게임이 아닙니다. 저희는 Last War Games, First Fun, Century Games 또는 그 어떤 파트너도 아닙니다. 내부자 접근 권한이 없습니다. 사전 패치 노트를 받지 않습니다. 어떤 식으로도 게임 퍼블리셔를 대표하지 않습니다.

모든 인게임 이름, 유닛 스탯, 스크린샷은 여러분이 게임 주변에서 기획하는 것을 돕기 위해 사용됩니다. 이는 미국 저작권법상 명시적 공정 사용입니다. 권리 보유자가 이의를 제기하시면 즉시 경청하겠습니다 — 위 이메일.

보안 태세 (저희가 건드리지 않는 것)

저희는 절대 게임 계정 정보를 요구하지 않습니다. 아이디도, 비밀번호도, 세션 토큰도. 계정에 로그인할 수 있는 어떠한 것도. r5tools 페이지가 그런 것을 묻는다면 탭을 닫고 이메일 주세요 — 저희 이름을 사칭한 피싱입니다.

자동화 없음

여러분을 대신해서 게임을 클릭하거나 탭하거나 무언가를 전송하지 않습니다. 저희 도구는 계산기이자 기획 도구이지, 봇이 아닙니다.

메모리 읽기 없음

여러분 기기의 게임 메모리, 네트워크 트래픽, 클립보드를 읽지 않습니다. 저희가 아는 모든 것은 여러분이 붙여넣거나 업로드한 데이터에서 옵니다.

이용약관 위반 없음

저희가 아는 한 LWS 이용약관을 위반하는 것은 없습니다. 저희는 기껏해야 UI가 있는 스프레드시트입니다.

사진은 로컬에 유지

프로필 스튜디오는 100% 브라우저에서 처리합니다. 사진은 저희 서버에 닿지 않습니다.

전송 중 암호화

모든 곳에서 TLS 1.2 이상. 액세스 코드는 평문이 아닌 해시로 저장됩니다.

언제든 삭제 가능

연맹 코드와 함께 이메일 주시면 30일 이내에 삭제합니다 — 보통 24시간 이내에.

데이터 처리 요약

  • 서비스 운영에 꼭 필요한 최소 데이터만 수집합니다.
  • 데이터를 판매하지 않습니다. 결코. 누구에게도.
  • 데이터를 광고에 사용하지 않습니다.
  • GDPR, CCPA, PIPA 정보주체 권리를 전 세계적으로 존중합니다.
  • 상세: 개인정보 처리방침.

버그 바운티 프로그램

보안 취약점을 발견하시면 인터넷에 알리기 전에 저희에게 알려주세요. 저희는 1인 개발이며 보안 보고를 진지하게 받아들입니다.

신고 방법

  • 제목 "Security"로 evanjones@mdr.net에 이메일.
  • 포함할 내용: 발견한 것, 재현 방법, (원하시면) 명예의 전당 등록용 이름/핸들.
  • 수정 확인 전까지는 공개 공개하지 말아주세요.

범위 내

  • r5tools.io 및 access-codes.r5tools.io의 인증 또는 세션 버그.
  • 다른 연맹의 업로드된 데이터를 읽는 방법.
  • 저희 오리진에서의 XSS, SQL 인젝션, SSRF, IDOR.
  • 액세스 코드 권한 확인 우회.

범위 외

  • 제3자 서비스(Stripe, Cloudflare, GitHub Pages, Railway는 벤더에 직접 신고).
  • PoC 없는 속도 제한 또는 무차별 대입 보고.
  • 익스플로잇 없는 누락된 보안 헤더.
  • 사용자 기기에 대한 물리적 접근이 필요한 취약점.
