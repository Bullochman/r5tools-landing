Terms of Service

Last updated: July 2026. If anything on this page conflicts with your use case, email hello@r5tools.io — we're small enough to actually read it.

1. Who runs this

r5tools.io is an independent tool suite for players of Last War: Survival (published by First Fun Group). We are not affiliated with, endorsed by, or licensed by First Fun Group or Last War: Survival. This is a fan-made toolkit.

2. What you're paying for (if you paid)

Access codes ($10 Personal, $30 Founding Alliance, $50 Alliance Bundle) unlock the suite on your device for 90 days per redemption. The Personal code is single-use; the Alliance codes are shareable within your alliance. Cost of goods sold is real (Claude API + hosting) — see the pricing page for the honest breakdown.

3. Refunds

  • Within 24 hours of purchase, no redemption: full refund, no questions asked. Email hello@r5tools.io.
  • Code already redeemed: no refund except in cases of tool downtime we caused (>48h continuous outage of the tool you bought for).
  • Alliance Bundle disputes: if your alliance disbands, we don't refund the R5 who purchased. Codes can be shared to a successor alliance.

4. Data we collect

Minimal:

  • Access-code usage (which code, when, from what tool) — for analytics + rate limiting.
  • IP address (masked to last-octet-x for analytics, full for rate limiting).
  • Browser language (for auto-toggling KR/EN).
  • Discord webhook URLs you paste for tool exports (stored per your alliance access code, never exposed to other users).
  • Roster CSVs you upload — stored per alliance signature, never shared with other alliances or third parties.

We do NOT collect: emails (unless you use the email-magic-link path), real names, in-game passwords, payment information (Stripe handles that directly). Full details in the Privacy Policy.

5. Rate limits

To keep the tool sustainable:

  • Roster Extractor: up to 30 extractions/month per alliance code (Bundle $30) or 60/month ($50). Personal $10 code: 20/month.
  • KB Chat: reasonable use — hundreds of questions per code per day is fine, thousands is not.
  • Support requests: 8 per hour per IP address to prevent spam.

If you hit a limit and think it's wrong, use the 💡 Feature Request button to tell us.

6. What we promise

  • The tools work when you sign in. If they're down we fix them.
  • Your roster data stays private — we don't sell it, share it, or use it for anything except the tool you uploaded it for.
  • New features ship regularly. The 💡 Feature Request button in every tool goes straight to Evan's Discord.

7. What we don't promise

  • That the game data in our KB is always current. LWS updates weekly; we catch up within 1-2 weeks.
  • That every tool works on every device / browser. We test Chrome + Safari on desktop + iOS. Firefox usually works. Older Android varies.
  • That we'll build every requested feature. High-vote-count requests get priority.

8. Prohibited

  • Do not use the tools to violate Last War: Survival's Terms of Service.
  • Do not upload rosters containing personal identifying information (real names, phone numbers, addresses). Player nicknames are fine.
  • Do not scrape or bulk-download the KB Chat responses. If you want a copy of the KB for offline reference, ask.
  • Do not resell access codes.

9. Changes

We may update these terms. Material changes get an in-tool notification (💡 banner at top) 7 days before taking effect. You can always find the current version at r5tools.io/terms.html.

10. Contact

Everything comes back to hello@r5tools.io or the 💡 Feature Request button in any tool.
