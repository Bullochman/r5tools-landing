Alliance Help daily task ladder (added 2026-07-16). Independent of the raw contribution-points ceiling, the Alliance Help daily task list unlocks progressive rewards at 10 / 50 / 100 / 200 / 300 cumulative helps in one day. The 300-help tier is the top rung of the daily task list, which implicitly puts the daily contribution-points cap at or above 300 helps (the last tier would not be reachable otherwise). Community guides publish this ladder as tier thresholds, not a cap statement, but the 300 upper tier is the tightest published lower-bound the KB now has on the daily-help ceiling. Downstream tools (Alliance Help Efficiency Tracker) can safely assume ≥300 helps/day as the plannable ceiling until an in-game screenshot pushes the number higher. [lastwartutorial.com Alliances page search-excerpt 2026-07-16 Tier B; tap-guides.com Last War Alliance guide 2025-10-24 WebFetch Tier C]

Additional gate confirmed 2026-07-15: only the top 32 alliances in the warzone are eligible to participate — even at ≥20 members and HQ 10+, an alliance ranked #33 or lower in server power is locked out until it climbs the leaderboard [treyexgaming.com/last-war-survival-alliance-duel-guide/ WebFetch 2026-07-15, Tier B; corroborated by lastwargame.online/en/game-alliance-duel/ search 2026-07-15]. This is a load-bearing gate for the R5-tools "who should we recruit next" question — top-32 threshold is the practical eligibility bar, not the ≥20-member floor.

Format. R4 or R5 places the Marshal's Guard platform on the world map. The platform is a 3×3-tile object placed at the anchor coordinate the leader picks (the working anchor for planning tools is {x:486, y:432} on many servers, but it's leader-controlled). Five Marshal's tanks appear on the platform; the alliance rallies them one at a time until either all five die or the 30-minute clock runs out.

  • Player HQ / city: 2×2 tiles per base footprint (per hive-planner tools; note some sources treat member "cells" as 3×3 in planner UIs where one cell = one base + 1-tile buffer).
  • Marshal's Guard platform: 3×3 tiles (1 event cell).
  • Alliance Center (world building placement varies; the on-base structure is 5-tile-footprint per planners): 5×5 tiles.
  • Missile silo (turret/rocket world object): 4×4 tiles.
  • Military Stronghold seasonal fortress: 21×21 tiles (a 7-cell × 7-cell block in a 3-tile-per-cell planner grid).

The overlap with game titles is common — e.g. the Warlord title (+5% MG rally, S6 Serpent Breath) is usually assigned to the same person who acts as War Coordinator. But they don't have to match: an R5 might title one R4 as "Warlord" for the S6 skill activation timing while giving the War Coordinator hat to a different R4 who is online during Enemy Buster. The LWS_Alliance_Ops_Center Ministry Assigner tool (planned per TOOLS_ROADMAP) separates the two dimensions explicitly.

Role tag convention. Some alliances prefix in-game display names with 3-letter role tags (RL-, RF-, GD-, GT-, SC-) so the R4 War Coordinator can filter the roster view instantly. Not enforced by the game — a Discord/spreadsheet convention that maps 1:1 to the table above.
