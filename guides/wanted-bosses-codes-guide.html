Wanted Bosses (Codes 39 / 64 / 87) — weekly-rotating hero-type-buffed boss damage race

Added 2026-07-15. Wanted Bosses are the game's rotating damage-race event in which a specific hero-type gets a +50 % damage buff. Sources: [theriagames.com/guide/last-war-survival-wanted-boss-code-39/ WebFetch 2026-07-15, Tier B]; [lastwarvault.com/guides/general/wanted-boss-guide/ WebFetch 2026-07-15, Tier B]; [lastwargame.online/en/bosses-code-39-64-87/ WebFetch 2026-07-15, Tier B]; [buffbuff.com Boss Code overview via search excerpt, 2026-07-15]; [btcpiratesnation.com/wanted-boss-code/ via search excerpt, 2026-07-15].

Weekly rotation (correction 2026-07-15 — prior KB claim that codes rotated daily was wrong; sources agree the rotation is weekly by day-of-week, two days per code, Sunday off):

Spawn schedule (per Vault + LastWar.online): 4 fixed windows per event day at 00:00 / 06:00 / 12:00 / 18:00 server time, each spawn staying on the map for 3 hours before despawning. Third-source corroboration 2026-07-16 via LastWar.online search-excerpt reconfirms the identical 4-spawn / 3-hour / 5-attempts / HQ-8 profile — the rotation and cadence are now triple-sourced.

Recommended squads by code:

CodeRecommended squad
Code 87 (Tank day)Kimberly, Mason, Williams, Marshall support
Code 64 (Missile day)Fiona, Swift, Tesla, McGregor; missile mono-formation
Code 39 (Aircraft day)DVA, Morrison, Schuyler front-loaded; skip tanks entirely

Fight mechanics:

  • Each attempt lasts 1 min 30 sec — inflict as much damage as possible in the window
  • Boss does not attack you — no defensive heroes needed; pure DPS squads win
  • 5 attempts per day; solo attacks only (no rally option per Vault)
  • Only your highest single-attempt damage counts for ranking rewards
  • Rewards structured as ranking rewards (top-N placement) + first-3-attacks individual rewards + 6-tier objective/goal rewards
  • Minimum HQ level 8 to participate

F2P optimization (per Vault):

  1. Always match troop type to the day — the +50 % type bonus vastly outweighs gear-quality differences
  2. Activate War Fever by scouting an enemy base before attacking for a free damage boost
  3. Use cheap attack + crit potions — meaningful damage gain vs their trivial cost
  4. Never skip a day — even a single attempt per spawn window guarantees baseline rewards; long-term compounding on Upgrade Ore + Synthetic Resin is significant
  5. Gear quality > hero star level for this event

R5 implication for tool design: since the buffed type rotates on a weekly Mon-Sat schedule (not daily), the Bosses Code Rotation Tracker tool from open-questions.md simplifies to a static 6-day calendar rather than a rolling day-N lookup — a fixed weekly-view widget is enough. Events F2P should hit every cycle (tier-1 priority):

  1. Wanted Bosses (Codes 39/64/87) — 5 free attempts/day, no cost, Upgrade Ore + Synthetic Resin compounds long-term. Never skip a day.
  2. Arms Race — free daily; hoard speedups + valor badges between cycles, dump them during matching phases only.
  3. Alliance Duel VS — Wednesday's valor-badge spend + Thursday's shard spend are the two biggest single-day contributions a free player can make. Chest 1–3 (Basic, 540k pts) is a realistic F2P weekly target.
  4. Ghost Ops — 3 assists + 5 plunders per Thursday hits 6 rewards with zero premium currency. Prefer Exclusive Weapon Shards over UR shards per LDShop — the weapon system is the F2P power multiplier that closes the gap with whales.
  5. Trade Truck plunders — 4 enemy trucks per day; six-refresh rule (refresh, plunder 1–2, wait 10–30 sec, refresh again — by the 6th refresh, a UR-quality truck is near-certain) [lw-spy.com/blog/last-war-truck-plunder-guide/ WebSearch 2026-07-15, Tier B]. Reindeer Sleds are the highest-priority truck type — Exclusive Weapon Shards are cited as the single most valuable reward in the entire truck system.
  6. Glittering Market — even without diamond spend, F2P earns ~300–360 Coins/6-day cycle from the daily-diamond exchange (only if diamonds are saved for it, not spent on shields). Coins do NOT expire between cycles. Stack 8–10 cycles = one full Mythic Gear Blueprint set (3,000 coins). This is the single most important long-cycle F2P strategy.
