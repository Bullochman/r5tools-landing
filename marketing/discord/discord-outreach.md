# Discord Outreach Targets — LWS Community

Real Discord communities where LWS R5/R4/officers actually hang out. Every entry below was verified by web research on 2026-07-17 — where a member count or invite link couldn't be confirmed, that's called out explicitly. **Do not invent numbers.** If an entry says "count unverified," it stays that way until Evan (or a bot) joins and reads the presence panel.

**One-line rule for all of these:** the r5tools.io mention is a footer, not the point of the post. Lead with value (a specific tool result, a screenshot, a math correction, a coord list), let the URL sit at the bottom. Discord mods nuke self-promo faster than Reddit — the "value-first" bar is even higher here than on r/LastWarSurvival.

**Free code across all posts:** `RONY-FREE` (Warzone 2007 native; works on any warzone). Alliance Founding tier $30 / Personal $10.

## Cross-cutting rules for Discord outreach

1. **Ask a mod before posting anywhere except channels explicitly labeled `#tools`, `#resources`, `#promotion`, or `#creator-projects`.** Discord culture varies wildly server-to-server — one server's `#general` welcomes it, the next server's bans on first strike.

2. **Never crosspost the same tool link to 3+ servers in a single hour.** Discord's anti-spam heuristics + user overlap (many R5s are in 5+ servers) will make you look like a bot.

3. **Screenshot > link when possible.** A 1080p PNG of the landing planner output posted inline drives more clicks than a bare URL. Load the PNG from `/screenshots/` in the r5tools-landing repo.

4. **When a mod DMs about promo, respond in <2 hours.** Acknowledge, ask which channels are OK, offer to remove if wrong. Never argue. The mod remembers your reply speed forever.

5. **After posting, engage for 60 min minimum.** Reply to every comment, even the low-effort ones. A post with 8 reply-threads outranks a post with 40 upvotes and no engagement in Discord's algorithm.

6. **Track: which server / which channel / which time / which post variant / clicks / DMs / conversions.** Same spreadsheet columns as `warzone-tracking.csv`. If a server produces 0 clicks in 72h, deprioritize.

7. **Never mention competitors by name.** Cpt Hedgehog, LW Spy, AMP, LW Atlas — all named above for research purposes only. In actual posts, position r5tools on its own merits.
