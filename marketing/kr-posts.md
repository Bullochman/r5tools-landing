**도구 8종 요약:**

1. **로스터 익스트랙터** (roster.r5tools.io) — 얼라이언스 멤버 리스트 스크롤 영상만 찍어서 올리면 이름/랭크/HQ/파워 전부 CSV로 뽑아줍니다. 한국어 닉네임 정상 인식되고, "Đ THOR G" 같은 특수문자 닉도 유니코드 정규화로 중복 제거합니다. 저희 94명 얼라 처음 뽑을 때 3분 걸렸습니다.

2. **하이브 그리드 매니저** (hive.r5tools.io) — MG 3×3 팩과 스트롱홀드 21×21 원형 배치 자동 생성. R5는 노랑, R4는 보라로 자동 색상 태그. 드래그로 위치 교환 가능하고 디스코드용 PNG 다운로드까지 원클릭.

3. **랜딩 플래너** — 시즌 2 시작 직후 알럴 퍼니스 주변에 누구를 어느 링에 놓을지 자동 배치. R5+타이틀 R4는 링 1, 미타이틀 R4는 링 2, R3 서포터는 링 3 이런 식으로 계급별 자동 정렬. 미니스트리 4자리 안 채워지면 경고 뜸.

4. **히트 시뮬레이터** — 시즌 2 블리자드가 지나갈 때 어느 셀이 -20°C 밑으로 떨어질지 시각화. 어니면 손실 걱정 없이 히터 배치 최적화 가능.

5. **동결 위험 대시보드** — 멤버별 히터 파워 vs 예상 최저 온도. 위험 등급 자동 계산. 시즌 2 워크 1~8 전 구간 커버.

6. **석탄 소모 계산기** (Coal Burn) — HHF 노멀/오버드라이브 소모율 정확히 계산 (KB에서 파낸 240 coal/min L30 기준). 오버드라이브 몇 시간 돌릴지 미리 예산 짜기.

7. **도시 점령 플래너** — 하루 2도시 캡 + 36시간 보호 창 지키면서 Wk.D 시나리오 자동 스케줄링.

8. **시즌 타임라인** — 8주 로드맵. 멤버별 체크박스로 진행률 표시. R5가 전체 진척도 실시간 확인 가능.

**왜 만들었냐:**

시즌 2 시작 전에 스프레드시트로 하이브 배치 짜본 R5분들은 아실 겁니다. 새 멤버 들어오면 다 다시 짜야 하고, 계급 바뀌면 또 다시 짜야 하고. 그거 하다가 정작 코울 예산은 못 짜서 워크 3에서 알럴 퍼니스 꺼트린 적 있습니다. 그런 일 다시 안 겪으려고 만든 도구들입니다.

## Post 7 — KR-tailored 기능 리스트 (한국 R5가 가장 관심 있어할 순서)

**대상:** 스크린샷 첨부 없이 텍스트로 신뢰 구축할 때, 또는 카페 공략글 형태로 게시

---

### 제목: 한국 R5분들이 가장 궁금해할 기능 위주로 정리 — r5tools.io

한국 알럴 특성상 가장 궁금해하실 순서대로 정리했습니다.

**1. 시즌 2 극지 폭풍 — 히트 & 동결 관리** ❄️
- 히트 시뮬레이터: 블리자드가 지날 때 어느 셀이 -20°C 밑으로 떨어질지 시각화. 워크별 온도 밴드 (-15/-30/-50°C) 반영.
- 동결 위험 대시보드: 멤버별 히터 파워 vs 예상 최저 온도. 위험도 자동 계산. "누가 얼어붙을지" 미리 확인.
- 왜 중요한가: 한국 알럴들이 시즌 2 실패하는 최대 원인이 히트 관리 실패. 도시 못 뺏겨도 얼라 퍼니스 꺼지면 끝.

**2. 하이브 팩 자동 배치 — MG + 스트롱홀드** 🐝
- MG 3×3 팩 자동 정렬 (게이트 방향, 워크백 시간 고려).
- 스트롱홀드 21×21 원형 배치 (알럴 센터 기준 방사형).
- R5=노랑, R4=보라, R3=파랑, R2/R1=회색 자동 색상. 한국 알럴이 선호하는 계급별 색상 코딩 그대로 반영.
- CSV 임포트로 94명 얼라 5초 안에 배치 완료.

**3. 시즌 타이밍 — 워크별 로드맵** 📅
- 8주 시즌 타임라인. 워크 1 랜딩 → 워크 3 첫 도시 → 워크 5 테슬라 어웨이크닝 → 워크 7 뉴클리어 퍼니스 → 워크 8 종료.
- 멤버별 체크리스트로 진행률 추적. R5가 알럴 전체 현황 한눈에.
- 시즌 3 골든 킹덤, 시즌 4 이버나이트 아일, 시즌 5 와일드 웨스트, 시즌 6 섀도우 레인포레스트 전부 지원.

**4. 로스터 익스트랙터 — 한국어 닉네임 완벽 인식** 📹
- 멤버 리스트 스크롤 영상 (30-90초) 업로드 → Claude Sonnet 비전으로 이름/랭크/HQ/파워 자동 추출.
- 한국어 닉네임 정상 인식. "Đ THOR G" / "Đ THOR G" / "Đ THOR Ğ" 같은 유니코드 변형 자동 병합.
- 파워 열은 K/M 단위 그대로 유지 (한국 알럴들이 선호하는 방식).
- CSV + JSON 다운로드. 다른 도구에 그대로 임포트 가능.

**5. 자원 관리 — 석탄 예산** ⛏️
- HHF (하이 히트 퍼니스) L30 노멀 = 240 coal/min, 오버드라이브 = 4배 = 960 coal/min 정확 반영.
- 오버드라이브 몇 시간 돌리면 얼마 필요한지 계산. 알럴 퍼니스 vs 개인 퍼니스 분리 예산 지원.
- KB에서 소스별 크로스 체크한 수치. 커뮤니티 추측치가 아닌 확정 데이터.

**6. 도시 점령 — Wk.D 스케줄러** 🏙️
- 하루 2도시 캡 + 36시간 보호 창 자동 관리.
- L1 (Wk1.D3) → L6 (Wk3.D6) 도시 언락 스케줄 반영.
- 히트/소일/근접성/도시 타입 가중 점수. 어떤 도시부터 뺏을지 자동 우선순위.

**7. 랜딩 플래너 — 시즌 시작 첫날 배치** 🎯
- 알럴 퍼니스 주변 링 배치. R5+타이틀 R4 = 링 1, 미타이틀 R4 = 링 2, R3 = 링 3.

- 시즌 시작 30분 전 스샷 뽑아서 알럴 채팅에 뿌리면 끝.

**8. 언어 & 데이터** 🌏

## Post 8 — KR-tailored FAQ

**대상:** 사이트 KR FAQ 페이지 겸 커뮤니티 게시글

### 제목: r5tools.io 한국 R5 자주 묻는 질문 (FAQ)

**Q1. 개인정보 안전한가요? 얼라 멤버 이름/파워 데이터 어디에 저장돼요?**

로컬 SQLite에 저장됩니다. 로스터 익스트랙터가 처리한 데이터는 사용자 브라우저 로컬 스토리지 + 서버 SQLite (roster.r5tools.io 컨테이너 내부)에 저장되고, 다른 얼라이언스와 공유 안 됩니다.

**중요:** 서버는 Railway 무료 티어라서 컨테이너 재시작 시 데이터 초기화됩니다. 그래서 **백업 버튼**을 만들어뒀습니다 — JSON 파일로 받아서 본인이 보관하세요. 다음번 접속 시 복원 버튼으로 복구 가능합니다.

앞으로 유료 티어로 전환하면 지속 저장(persistent volume)으로 옮길 예정입니다.

---

**Q2. 진짜 무료예요? 나중에 돈 뜯어가지 않나요?**

지금 시점 (2026년 7월)에 무료 코드 `RONY-FREE`로 8개 도구 전부 열립니다. 코드는 만료 없음.

**미래 계획:** 나중에 유료 티어 만들 예정 ($10 개인 / $50 알럴 번들 / $5 로스터 익스트랙터 프리미엄). 하지만:
- 워존 2007 알럴은 영구 무료.
- 한국 알럴은 우선 무료 유지 계획. 왜냐하면 한국 R5분들 피드백이 도구 개선에 제일 도움 됐고, 앞으로도 그럴 것이기 때문입니다.
- 유료 티어 시작해도 이미 발급된 `RONY-FREE` 코드는 계속 유효.

**결론:** 지금 무료로 쓰시고, 나중에 유료 티어 나오면 그때 결정하시면 됩니다.

---

**Q3. First Fun 공식 도구인가요? 계정 정지 안 당해요?**

**아닙니다. 공식 아닙니다.** First Fun과 무관한 팬 프로젝트입니다.

**계정 정지 안 당합니다.** 이유:
- 게임 클라이언트에 개입 안 함. 스크린 영상 업로드 → 이미지 인식일 뿐.
- 게임 API 호출 안 함.
- 자동화(오토) 아님. R5가 직접 도구로 계획 짜고 게임에서 수동 실행.
- 스프레드시트로 계획 짜는 것과 동일한 카테고리.

Whiteout Survival 계열 도구 (Q1/2025 대량 사용됨)와 동일한 접근 방식입니다.

---

**Q4. 한국 워존에서도 작동하나요? 워존 번호 몇 번인지 상관 있어요?**

작동합니다. 워존 번호 상관 없습니다.

**설명:**
- 각 도구는 워존별 시즌 타이밍만 반영. 예: 워존 2007은 시즌 2 워크 3, 워존 900은 시즌 3 워크 5.
- 상단 워존 선택기에서 본인 워존 번호 입력하면 그에 맞는 시즌/워크 자동 반영.

**Q5. 왜 코드 이름이 RONY-FREE예요?**
