# r5tools.io — TikTok scripts (10 × 30s)

R5-to-R5 voice. Every number in these scripts is sourced from `LWS_Knowledge_Base/kb/06-season-2-frozen.md` — no invented mechanics. If a script cites a number, it is a real game number.

Reusable disclaimers:
- "R5" / "R4" refer to Last War: Survival alliance ranks. Voiceover should say "R-five" and "R-four" not "arr five."
- CTA base: **"Free at r5tools.io. Code RONY-FREE in bio."** Variants noted per-script.
- No emojis in voiceover. Emojis fine in on-screen text overlays.
- Total runtime for each script sits at ~28-30s. Hook is 0-3s, body 4-25s, CTA 26-30s.
