**제목: 우리 워존용 계획 도구 11종 만들었습니다 — 어느 얼라이언스든 무료, R5·임원분들 자유롭게 뜯어보세요**

안녕하세요 — 워존 2007 RONY R5 Evan(에반)입니다. 스튜디오도 아니고 마케팅 팀도 아니고, 그냥 시즌 2 폴라 스톰 시작 전날 밤 스프레드시트 6개 켜놓고 94명 알럴 퍼니스 배치하다가 "2026년에 왜 이걸 손으로 하고 있냐"고 자문한 R5입니다. 그래서 하나 만들었습니다. 그다음 하나 더. 결국 11개가 됐고, WZ 2007 이웃 R5분들이 접근권 달라고 해서 오늘부로 모든 얼라이언스에 열었습니다.

**도구가 실제로 하는 일.** **랜딩 플래너**는 로스터 (로스터 익스트랙터에서 뽑거나 수동 입력)를 받아서 랭크 + 타이틀 순으로 정렬하고, R5 + 타이틀 R4를 알럴 퍼니스 링 1에, 미타이틀 R4를 링 2에, 핵심 R3 서포터를 링 3에, 나머지를 외곽 링에 배치합니다. 미니스트리 자리가 안 채워지면 경고 뜹니다 (랜딩 당일에 흔한 실수 — 맵 도착해서 워로드 타이틀 아무도 없는 걸 그때 깨달음). **히트 시뮬레이터**는 블리자드 경로를 얼라이언스 그리드에 겹쳐서 −70°C 동결 임계값 밑으로 떨어지는 셀을 빨갛게 표시합니다 — 블리자드 오기 **전에** HHF 배치 옮길 수 있습니다. **석탄 소모 계산기**는 R5분들이 자주 틀리는 HHF 오버드라이브 계산을 정확히 해줍니다: HHF L30 노멀 소모 약 240 coal/min, 오버드라이브는 약 4배, 50명 얼라이언스가 워크 4 심각한 블리자드 중 동시 오버드라이브 돌리면 3시간에 약 860만 석탄 소모됩니다. 이 숫자는 대부분의 R5가 처음 볼 때 놀랍니다. **도시 점령 플래너**는 하루 2도시 캡 + 36시간 보호 창을 자동 반영해서 실수로 보호 중인 도시 재공격 안 하게 해줍니다. **로스터 익스트랙터**는 얼라이언스 멤버 리스트 화면 녹화를 OCR로 처리해서 약 30초에 CSV로 뽑아드립니다 — 유니코드 이름 정상 처리 (한국어, 일본어, 특수문자 알럴 태그 전부), Ð / ø / Đ 같은 유사 문자 포함 이름 자동 중복 제거.

**무료 티어는 진짜 무료입니다.** 랜딩 플래너, 히트 시뮬레이터, 석탄 소모 계산기, 도시 점령 플래너, 시즌 타임라인, 하이브 그리드 매니저 — 전부 코드 없이 이메일 없이 계정 없이 됩니다. 로스터 익스트랙터 무료 티어는 CSV로 뽑아주고, 유료 티어는 히스토리 추적 (얼라이언스 파워 시간별 라인 차트) + 디스코드용 PNG 익스포트가 추가됩니다. 워존 2007 얼라이언스는 이웃이니까 코드 `RONY-FREE` 로 유료 기능도 무료입니다. 다른 워존 얼라이언스도 런치 기간에는 `RONY-FREE` 로 무료로 쓸 수 있습니다 — 의미 있는 만큼 계속 열어둘 예정입니다. **얼라이언스 창립 티어는 $30, 얼라이언스 전체 평생, 첫 10개 얼라이언스만** — 이후 $50입니다. 개인 티어는 R5 1인 계정 $10입니다.

**언어 지원.** WZ 2007이 한국 얼라이언스 위주라 한국어 우선 설계로 만들었습니다. KR R5분들이 파파고로 영어 도구 복붙하는 걸 보다가 답답해서요. 완전한 네이티브 로컬라이제이션: **한국어 / English / Español / Português / 日本語 / Deutsch / Français**. 상단 우측 언어 드롭다운, UI + 모든 게임 용어 네이티브 언어로. 자동 번역 아닙니다 — 모든 문자열을 해당 워존 네이티브 스피커가 검수했습니다. 번역 이슈 발견하시면 DM 주시면 당일에 수정합니다.

**r5tools가 아닌 것.** 맵 스캐너 아닙니다. 게임 계정에 로그인하는 봇 아닙니다. 상대 얼라이언스 로스터 긁는 도구 아닙니다. 여러분이 "PNG 다운로드" 눌러서 직접 붙여넣기 전에는 디스코드에 아무것도 자동 게시 안 합니다. 로스터 데이터는 여러분 기기의 로컬 SQLite에 저장됩니다 — JSON으로 백업하고 언제든 삭제 가능. 무료 티어는 계정 생성 없음. 이메일 요청은 얼라이언스 창립 티어 매직 링크 로그인 때뿐이고, 그것도 코드 기반 무료 티어만 쓰실 거면 필요 없습니다.

**링크 + 부탁 하나.** **r5tools.io** — 지금 `RONY-FREE` 로 30초면 도구 써보실 수 있습니다. R5이신데 시간 절약 되신다 싶으면 다른 R5 한 분께 알려주세요. 버그 있으면 이 스레드에 답글 남기시거나 DM 주세요 — 지원 티켓 돌리기 없이 제가 직접 고칩니다. 좋은 사냥 되세요.
