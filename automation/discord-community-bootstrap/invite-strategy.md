# Invite Strategy — Growing r5tools Community Discord

Small-server growth is a distribution problem, not a marketing-copy problem. Every touch point where a user is already engaged with r5tools is a natural invite surface. Plaster the link across all of them; measure which one converts, double down.

## The invite link

**Once server exists:** Server Settings → Invites → **New Invite** → set:
- Max age: **Never expire**
- Max uses: **Unlimited**
- Grant temporary membership: **OFF**
- Channel: `#welcome`

Copy the resulting `discord.gg/xxxxxx` URL. Once you have Level-3 boosts, reserve the vanity `discord.gg/r5tools` and swap everywhere below.

**Interim/permanent invite URL slot:** save it in `bot-state.json` under `invite.public_url` so all downstream scripts and future automation can reference one canonical source.

## Anti-patterns

- **Don't** buy Discord invite ads / server-list promotion. Low-quality inbounds pollute the R5-to-R5 tone the server is built on.
- **Don't** cross-promote in someone else's server without their R5's permission. Same rule as `discord-outreach.md`.
- **Don't** DM invites unsolicited. Ever. Kills your reputation and Discord anti-spam flags it.
- **Don't** set the invite as temporary — temp members are auto-kicked on disconnect, which breaks the "come back after the blizzard" pattern.

## Measurement

Track invite conversion in `bot-state.json` → `invite.metrics`:

- **Landing footer clicks** — analytics.js already tracks `discord_footer_click` events; aggregate weekly.
- **Cheat-sheet QR scans** — QR URL with `?src=cheatsheet` param, count in analytics.
- **Email footer clicks** — `?src=email_<template_name>` param.
- **Reddit / X bio** — invite URL variants like `discord.gg/r5tools-rx` if you can generate distinct vanity variants later.

Weekly review: which surface produced the most (a) invites accepted, (b) rules-accepted conversions, (c) retained members after 7 days. Kill the losers, double the winners.

## First-100-members milestones

- **10 members:** private launch, R5-neighbors of RONY only, iron out setup bugs.
- **25 members:** invite Founding-tier R5s + WZ 2007 alliances that already know the tools.
- **50 members:** enable in landing footer and email footer.
- **100 members:** enable **Server Discovery** in Server Settings; add discovery tags: *Last War Survival*, *Alliance Management*, *R5 Tools*. Free discoverable traffic starts here.
- **250 members:** start pushing on TikTok / Reddit / cross-Discord outreach.
- **1000 members:** community is self-sustaining; Evan can pull back to answering `#tools-help` questions + Founding-tier private channel.
